# 打包说明

## 打包完成！

发布包已成功创建：`StockAnalysisSystem-v1.0.0-Release.zip`

## 包含文件

```
StockAnalysisSystem-v1.0.0-Release.zip
├── StockAnalysisSystem.UI.exe        # 主程序（包含所有依赖，约180MB）
├── appsettings.json                  # 配置文件（需用户修改）
├── appsettings.example.json          # 配置示例
├── init_database.sql                 # 数据库初始化脚本

```

## 分发给用户

用户只需要：
1. 解压 zip 文件
2. 修改 `appsettings.json` 配置数据库
3. 运行 `init_database.sql` 初始化数据库
4. 双击 `StockAnalysisSystem.UI.exe` 启动

## 用户使用环境要求

- Windows 10 或更高版本
- MySQL 5.7+ 或 MariaDB 10.2+
- 无需安装 .NET 运行时（已包含在 exe 中）

## 重要提示

已将 `appsettings.json` 中的敏感信息（数据库密码、API密钥）替换为占位符，可以安全地分发给他人。

## 下次更新

如需重新打包：
```powershell
cd "d:\工作\StockAnalysisSystem\StockAnalysisSystem.UI"
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -o "../publish-release"
Compress-Archive -Path "../publish-release/*" -DestinationPath "../StockAnalysisSystem-v1.0.0-Release.zip" -Force
```
