# 首板后回落策略说明

## 策略概述

首板后回落策略是一种基于涨停板的选股策略，通过识别股票首次涨停（首板）后的回落机会来进行买入。

## 策略逻辑

### 1. 识别首板
- **从行情表（stockdailydata）中识别**，不使用涨停表
- 遍历历史数据，找到第一次涨停（涨幅 >= 9.95%）
- 记录首板的日期和最低价

### 2. 寻找回落机会
- 在首板后的 3-30 天内寻找买入机会
- **关键条件**：股价必须回落到首板最低价附近
  - 允许略低于首板最低价：最多低于 5%
  - 允许略高于首板最低价：最多高于 2.5%
  - **不允许在远高于首板最低价的位置买入**（确保是真正的回落）

### 3. 评分机制
- 根据距离首板的天数计算评分
- 天数越少，评分越高（0.5-1.0 分）
- 公式：`评分 = 1.0 - (天数 / 最大天数) * 0.5`

## 策略参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| LimitUpThreshold | decimal | 9.95 | 涨停阈值（%） |
| PullbackRange | decimal | 0.05 | 回落范围（允许低于最低价5%，或高于最低价2.5%） |
| MaxDaysAfterLimitUp | int | 30 | 首板后最大有效天数（天） |
| MinDaysAfterLimitUp | int | 3 | 首板后最小等待天数（天） |

## 选股条件

1. 股票存在首板记录
2. 距离首板日期在 3-30 天内
3. **当前收盘价回落到首板最低价附近**：
   - 允许略低于首板最低价（最多 -5%）
   - 允许略高于首板最低价（最多 +2.5%）
   - **不允许远高于首板最低价**（确保是真正的回落）
4. 每个交易日最多发出一个信号

## 买入信号示例

```
# 符合条件的买入信号（回落到首板最低价附近）
首板后回落, 首板日期:2024-03-15, 首板最低价:10.50, 距首板7天, 当前价:10.35, 偏差:-1.43%
首板后回落, 首板日期:2024-03-15, 首板最低价:10.50, 距首板10天, 当前价:10.65, 偏差:+1.43%

# 不符合条件的信号（价格远高于首板最低价，未真正回落）
当前价:11.20, 偏差:+6.67%（不符合：远高于最低价+2.5%，不是回落）
```

## 策略优势

1. **风险控制**：选择首板后的回落点，避免追高风险
2. **时间价值**：距离首板越近，评分越高，捕捉短期机会
3. **技术支撑**：首板最低价通常具有较强的支撑作用
4. **灵活性**：可调整回落范围和时间窗口，适应不同市场环境

## 策略风险

1. **首板陷阱**：虚假首板后可能持续下跌
2. **时间滞后**：3天的最小等待期可能错过部分机会
3. **波动风险**：回落到支撑位后可能继续下跌
4. **市场环境**：不同市场环境下有效性差异较大

## 使用建议

1. **市场环境**：适合震荡市或温和上涨的市场
2. **参数调整**：
   - 牛市：可适当缩短 MaxDaysAfterLimitUp
   - 熊市：应扩大 PullbackRange，等待更深回落
   - 震荡市：使用默认参数即可
3. **组合使用**：建议与均线策略、量能策略组合使用，提高准确率
4. **止损设置**：建议设置 5-8% 的止损位

## 回测建议

建议对以下参数组合进行回测：

1. **保守型**：PullbackRange=0.08, MinDaysAfterLimitUp=5
2. **平衡型**：PullbackRange=0.05, MinDaysAfterLimitUp=3（默认）
3. **激进型**：PullbackRange=0.03, MinDaysAfterLimitUp=1

回测周期建议至少 1 年，以覆盖不同市场环境。
